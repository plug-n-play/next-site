## useRouter

As you can see, [useRouter](https://nextjs.org/docs#use-router) allows you to access the `router` object inside the page, it's a [React Hook](https://reactjs.org/docs/hooks-intro.html), and it works with functional components.

In the following example you can see `useRouter` being added to a component other than the exported page:

```jsx
import { useRouter } from 'next/router';
import Layout from '../components/MyLayout.js';

const Content = () => {
  const router = useRouter();
  return (
    <>
      <h1>{router.query.title}</h1>
      <p>This is the blog post content.</p>
    </>
  );
};

const Page = () => (
  <Layout>
    <Content />
  </Layout>
);

export default Page;
```

export default ({ children }) => <Layout meta={meta}>{children}</Layout>;
