## Setup

Create a folder named `api` in `pages`, a sub-folder named `randomQuote` and a file `pages/api/randomQuote/index.js`.

```bash
export default (req, res) => {
  res.status(200).json({
    'quote': 'Write tests, not too many, mostly integration',
    'author': 'Guillermo Rauch'
  })
}
```

Now open your browser and try accessing localhost:3000/api/randomQuote
